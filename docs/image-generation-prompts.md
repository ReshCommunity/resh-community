## Quick Batch Execution Script

Create a shell script `batch-generate.sh`:

```bash
#!/bin/bash
# Change to script directory
cd ~/.claude/skills/gemini-image-generator/scripts

# Array of commands (add subset based on quota)
commands=(
  # BATCH 1: Crypto Basics (4-12)
  './venv/bin/python ./generate.py --subject "Cryptocurrency beginner guide with friendly welcome icons" --slug "cryptocurrency-for-beginners" --category crypto-basics'
  './venv/bin/python ./generate.py --subject "Crypto glossary dictionary with word bubbles" --slug "crypto-terminology-glossary" --category crypto-basics'
  './venv/bin/python ./generate.py --subject "Network nodes distributed showing decentralization" --slug "decentralization-explained" --category crypto-basics'
  # ... add more as needed
)

# Execute each command
for cmd in "${commands[@]}"; do
  echo "Running: $cmd"
  eval $cmd
  echo "Waiting 2 seconds..."
  sleep 2
done
```
