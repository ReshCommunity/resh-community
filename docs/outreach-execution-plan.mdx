## Phase 3: Email Sending Process

### For Each Email

**Step 1: Pre-Send Check**
- [ ] Verify email address is current
- [ ] Visit their website to confirm it's still active
- [ ] Check if they have a specific person to contact
- [ ] Review and personalize the email template
- [ ] Test all URLs in the email

**Step 2: Send**
- Copy email template
- Replace any placeholders
- Add any personalization notes
- Check for typos
- Send

**Step 3: Post-Send**
- Log in tracking spreadsheet with date sent
- Add to email label/folder
- Set follow-up reminder (Day 4 and Day 10)
- Note any special circumstances

## Phase 4: Response Handling

### Positive Response Workflow

**"Yes, we'll add it"**
1. Send thank you email immediately
2. Ask for expected live date
3. Schedule follow-up to confirm link is live
4. Update tracking: "Link Acquired"
5. Verify link when live
6. Update tracking: "Link Live"

**"Send more information"**
1. Provide requested details promptly
2. Offer custom content if appropriate
3. Set follow-up reminder (7 days)
4. Continue nurturing relationship

**"We'll review it"**
1. Send thank you note
2. Set reminder (14 days)
3. Offer to answer questions during review

### Negative Response Workflow

**"Not for us"**
1. Accept graciously
2. Ask for feedback (if not provided)
3. Thank them for their time
4. Update tracking: "Declined"
5. Note reason for future reference

**"Too many requests"**
1. Acknowledge respectfully
2. Offer custom content instead
3. Note to revisit in 6+ months

### No Response Workflow

**After 3 emails (Day 10 follow-up)**
1. Mark as "No Response"
2. Archive for now
3. Note to revisit in 6+ months with new content

- Acquire 2-3 links

**Month 1 (Complete Follow-ups)**
- Complete all Day 4 and Day 10 follow-ups
- Achieve 25% overall response rate (~9 responses)
- Acquire 4-6 links

**Quarter 1 (Ongoing)**
- All follow-ups complete
- 10+ links acquired
- Relationships built for future outreach

### Success Indicators

**Green Flags (Keep Doing)**
- High response rates from certain DA ranges
- Specific templates getting better responses
- Certain types of sites more receptive
- Patterns in positive responses

**Red Flags (Adjust)**
- Consistent rejections with same reason
- Very low response rates
- Emails bouncing
- Wrong contact information
